# Omniwrap - A Package Manager wrapper

#### _Please keep in mind that Omniwrap is still heavily in development therefore a lot of the expected features might be missing as of right now._

### What does it do?
Omniwrap helps you use different package managers on multiple supported distro's. It does so by providing you with simple commands so you don't have to memorize different ones for each package manager.

### Omniwrap currently supports these package managers:
- pacman
- zypper
- homebrew - Only tested on MacOS
- xbps

### Following package managers are expected to be supported in the future:
- dnf
- apt
- eopkg

### How to install?
These are the only ways to currently install Omniwrap that are confirmed to work
- Through the AUR:
   - ``` yay -S omniwrap ```
- Installing as a package
   - ``` git clone https://github.com/voidVoy7/Omniwrap.git ```
   - ``` cd *wherever you cloned it to* ```
   - ``` makepkg -si ```

### How to use?
You can use Omniwrap by simply typing ``` ow ``` in your terminal and you will be shown a list of all available commands for each package manager, after that its just a matter of looking at the right list.
